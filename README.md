# Interior Designer AI
>Generate interior design renders with DallE

![9RAdS.png](https://cdn.kevingil.com/thumbnail_image.png)



## Setup

```sh
from client/
npm install

from server/ 
pip install -r requirements.txt
```


## Run client

```sh
npm run dev
```


## Run server
```sh
python server.py
```

## Build with Docker
```sh
docker compose build
docker compose up -d
```
